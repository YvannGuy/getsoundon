# Modules

This directory contains domain-oriented modules for the GetSoundOn migration.

- `users`: profile and role logic
- `listings`: catalog, search, listing read/write
- `bookings`: request/offer/booking lifecycle
- `messaging`: conversation and message flows
- `payments`: Stripe checkout/webhook/payout
- `reviews`: ratings and review lifecycle
- `disputes`: dispute opening and resolution workflow
- `shared`: cross-domain types and utilities
